# 求是基金App

## 项目简介
求是基金是一款智能基金分析助手，基于现代Android开发技术栈，提供专业的基金分析和投资建议。

## 核心功能
- 📱 基金搜索与分析
- 📊 单基金深度分析
- ⚖️ 多基金对比
- 🎯 基金组合配置
- 🔎 市场扫描
- 🤖 多Agent专业分析

## 技术栈
- **语言**: Kotlin
- **UI框架**: Jetpack Compose + Material 3
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **网络请求**: Retrofit
- **数据库**: Room
- **图表库**: Vico

## 智能分析算法
- AHP层次分析法
- 模糊数学综合评判
- 贝叶斯概率预测
- 多因子评估体系（41个因子）

## 数据来源
- 天天基金网API
- OpenAI/通义千问API（LLM分析）

## 安装说明
1. 克隆仓库：`git clone https://github.com/qingjian0/fund-analysis-app.git`
2. 使用Android Studio打开项目
3. 同步Gradle依赖
4. 构建并运行应用
